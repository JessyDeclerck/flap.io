# FLAP.io by J.DECLERCK and S.GRULL ### Guide utilisateur <ul> <li style="color:blue;font-size:120%;">Prérequis</li> <br> Afin de faire fonctionner FLAP.io, il faut avoir installer le logiciel "node js".<br> Il est disponible à cette adresse <a href="https://nodejs.org/en/download/" target="_blank">Node js</a><br><br> <li style="color:blue;font-size:120%">Installation</li> <br> <ul> <li> Télecharger le fichier archive flap.io <a href="https://nodejs.org/en/download/" target="_blank">flap.io</a> </li> <li> Dézipper le fichier archive flap.io sur votre disque dur.</li> <li> L'arborescence des répertoire sera alors :<li> <ul> <li>+flap.io</li> <ul> <li>+models</li> <li>+models</li> <li>+node_modules</li> <li>+public</li> <li>+view</li> </ul> </ul> <li> Pour lancer le serveur, à l'aide de la console, se placer dans le répertoire flap.io<br> et tapper la commande "node flapServer.js".</li> <li>Une fois le serveur lancé :</li> <ul> <li>Ouvrir votre navigateur internet.</li> <li>Se connecter à l'adresse localhost:8095</li> <li>Enjoy</li> </ul> </ul><br> <li style="color:blue;font-size:120%">Comment jouer</li> <br> <ul> <li>Une fois arrivé sur la page de connexion :</li> <ul> <li>Vous avez un compte, rentrez alors votre pseudo et votre mot de passe puis cliquez sur le bouton "login"</li> <li> Vous n'avez pas de compte, cliquez sur le lien "inscrivez-vous"</li> </ul> <li>Sur l'écran de <b>création de compte</b>, saisissez un login et un mot de passe puis licquez sur le bouton "Création compte"</li> <li> Une fois connecté et sur la page de bienvenue, vous avez le choix :</li> <ul> <li>De rejoindre une partie de jeu.</li> <li>De vous deconnecter et de retourner à l'écran de connexion.</li> </ul> <li>Jouer à flap.io</li> <ul> <li>L'écran d'acceuil vous averti du nombre de joueurs sur le serveur, un minimum de deux joueurs est requis pour débuter une partie.<br> une fois le nombre de joueur requis, le jeu débutera automatiquement à l'issue d'un compte à rebours de 10 secondes</li> <li>Le but du jeu est simple : Aller le plus loin possible en passant par les trous dans les murs. Le controle du bird se fait avec la touche espace. une pression fera monter le bird.<br></li> <li>La partie est terminée dès que vous touchez un mur ou sortez de l'écran par le haut ou par le bas.<br> Une fois sur l'écran de fin, il faudra attendre que les autres joueurs perdent pour recommencer la partie.</li> </ul> </ul> </ul> <a href="index.html" target="_blank">Retour vers le sommaire</a>
